Simon Game

VHDL memory game for FPGAs.

Simon Game is a VHDL memory game that displays incremental random sequences on a LED Matrix by creating a finite state machine and implementing RAM and ROM models.

Also, it incorporates a binary to BCD to 7-segment conversion to visualize the current level on the FPGA 7-seg Display.

  • Date: May 2019
  • Category: VHDL
  • Core skills: VHDL and RTL Design, Nexys 4 DDR FPGA, Xilinx ISE, Test Benches, IP Core, Git.

Power Distribution PCB

Redesigned and assembled a Power Distribution 4-layer PCB that controlled the power from the battery to all the electrical devices on the racecar.

Delivered up to 12 Amps to each of the 11 outputs with minimum temperature rise, through an in-depth study of PCB layout, trace width and power and ground planes.

Added current and voltage limiting circuits to protect the MSP430 inputs by using Zener diodes and resistors.

Integrated current sensing circuits to collect output power data and send it to a Datalogger via CAN bus.

  • Date: May 2017 - September 2018
  • Category: PCB Design
  • Core skills: Altium Designer, Analog Design, C programming in Code Composer Studio, Saturn PCB Toolkit, MSP430 Microcontroller, SMD soldering, Oscilloscope, Multimeter.
